# Support

Use GitHub Issues for:

- Reproducible bugs.
- AEDT/PyAEDT compatibility findings.
- Solver-specific workflow requests.
- Documentation corrections.

Include:

- Operating system.
- AEDT version.
- PyAEDT version.
- MCP client and transport.
- Exact command or tool call.
- Relevant logs with license keys, hostnames, and private paths redacted.

Use Discussions for ideas, recipes, examples, and workflow design.
